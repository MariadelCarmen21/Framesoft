Listas Desplegables

Descripcion

Una lista desplegable HTML, o combo box en los lenguajes de programación, permiten a los usuarios de una página web, seleccionar un valor de una lista de opciones, como las que observamos cuando nos piden ingresar nuestro país en alguna pagina. Para crear una lista se usa la etiqueta SELECT, que usualmente debe llevar un nombre y un ID, el nombre se usa para que el programa que procese los datos del formulario pueda saber qué información está recibiendo. El ID se usa para poder acceder a ellapor medio de JavaScript, por ejemplo para validar si se ha llenado, para agregar o eliminar opciones, etc.

Ejemplos:


                

                         <select name="color" id="color"> 
                         <option value="r"Rojo/option >
                         <option value="a"Azul/option > 
                         < option value="v"Verde/option >
                         <select >

Con el código anterior se crea una lista llamada color, con un id nombrado también como color. La lista posee 3 opciones, estas fueron agregadas con las etiquetas <option/option>. Como se puede observar hay un atributo llamado value en cada opción, este es el valor que llega al programa que procesa el formulario, por ejemplo, si alguien selecciona el color rojo en esta lista, el programa que procese la información recibirá una variable llamada color con el valor r. La descripción de cada valor de la lista es el texto se coloque entre las etiquetas <option/option> Si se agrega el código anterior en una página web, el resultado será como el del ejemplos. 



              <select name="OS">
                <option selected value="0" Elige una opción /option>
                    <optgroup label="Microsoft"> >
                    <option value="1">Windows Vista/option> 
                   <option value="2">Windows 7/option> 
                    <option value="3">Windows XP/option> 
               </optgroup>
                <optgroup label="Linux"> 
                   <option value="10">Fedora/option> 
                    <option value="11">Debian/option> 
                    <option value="12">Suse/option> 
                </optgroup>
            </select>

Con la etiqueta <optgroup> se pueden crear grupos de opciones para así clasificarlos 



              <select name="OS" size=9>
              <option selected value="0"> Elige una opción /option>
              <optgroup label="Microsoft"> 
              <option value="1">Windows Vista/option> 
              <option value="2">Windows 7/option> 
              <option value="3">Windows XP/option> 
              </optgroup>
              <optgroup label="Linux"> 
              <option value="10">Fedora/option> 
              <option value="11">Debian/option> 
              <option value="12">Suse/option> 
              </optgroup> 
              </select>

Si lo que se desea es que esta no sea una lista desplegable, entonces se genera un código indicando el size de la lista
