Navegación

Descripcion

La navegación disponible en Bootstrap comparte marcas y estilos generales, desde la clase base .nav hasta los estados activo y deshabilitado. Cambia las clases de modificadores para cambiar entre cada estilo. El componente base .nav está construido con flexbox y proporciona una base sólida para construir todo tipo de componentes de navegación. Incluye algunas anulaciones de estilo (para trabajar con listas), algo de relleno de enlaces para áreas de acceso más grandes y estilo básico deshabilitado.

Ejemplos:

    Active Link Link Disabled 

                

                     < class="nav">
                        <  class="nav-link active" aria-current="page" href="#">Active>
                        <  class="nav-link" href="#">Link>
                        <  class="nav-link" href="#">Link>
                        < class="nav-link disabled">Disabled>

Use ul , ol si el orden de sus elementos es importante, o arroje el suyo propio con un elemento nav. Debido a que .nav usa display: flex, los enlaces de navegación se comportan igual que los elementos de navegación, pero sin el marcado adicional. Para transmitir el estado activo a las tecnologías de asistencia, use el atributo aria-current: use el valor de página para la página actual o verdadero para el elemento actual en un conjunto. Las clases se utilizan en todo momento, por lo que su marcado puede ser súper flexible.

  • Inicio
  • Sobre nosotros
  • Contacto 


              <nav class="menu">
              <ul>
              <li><a href="#">Inicio</a></li>
              <li><a href="#">Sobre nosotros</a></li>
              <li><a href="#">Contacto</a></li>
              </ul>
              </nav>

En este ejemplo, un bloque <nav> es usado para contener una lista no ordenada (<ul>) de enlaces. Con el CSS apropiado, esto puede ser presentado como una barra lateral, una barra de navegación o un menú desplegable.

  • Curso de Html5
  • Curso de CSS
  • Curso de Javascript


              <nav>
                
    
                  <li><a href="cursohtml.html">Curso de Html5</a></li>
                  <li><a href="cursocss.html">Curso de CSS</a></li>
                  <li><a href="cursojavascript.html">Curso de Javascript</a></li>
                  </ul>
              </nav>

Ésta etiqueta es para enlaces que favorezcan la navegación por la web, que nos dirijan a zonas importantes.
